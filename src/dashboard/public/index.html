⬆️ رفع كوكيز جديدة
✅ لا يحتاج m_sess — ارفع كوكيز facebook.com العادية وسيعمل البوت بـ HTTP Long-Poll تلقائياً.
🔄 تغيير فوري: بعد الحفظ سيتم تغيير حساب البوت تلقائياً دون إعادة تشغيل.
📋 JSON Array — مصفوفة من الكوكيز (c3c / Cookie-Editor / EditThisCookie)
[{"key":"c_user","value":"123..."},{"key":"xs","value":"abc..."},{"key":"datr","value":"xxx"}]
أو: [{"name":"c_user","value":"..."}] — يدعم name و key
🔤 نص الكوكيز — انسخ مباشرة من المتصفح (DevTools → Network → Request Headers → Cookie)
c_user=123456; xs=abc:def; datr=xxxx; sb=yyyy; fr=zzzz
🔑 توكن Facebook — الصق توكن EAAAA...
سيتم تحويل التوكن تلقائياً إلى كوكيز وتسجيل الدخول
EAAAA6z8X4Ba...
🦊 Netscape Format — صيغة كوكيز المتصفح القديمة
# Netscape HTTP Cookie File
facebook.com TRUE / FALSE 0 c_user 123456
